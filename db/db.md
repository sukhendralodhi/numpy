# Types of data
1. Quantitative Data
Quantitative data is a numerical and can be measured objectively. This data is often used for statical analysis and modeling.

A. Expressed in numerical form
B. Example - weight, volume, cost of an item.

2. Qualitative Data

Qualitative data is descriptive and non-numerical, offering insight into attributes and charactersitics.

A. Descriptive not numerical
B. Example - Name, gender, haor color of a person.
